Curfree 是一个免费工具，专门用于解决 Cursor 编辑器的 TooMany 机器码问题。

主要功能：

- 重置机器码 ID（最右侧按钮）
- 刷新数据（倒数第三个按钮）
- 主动数据备份（倒数第二个按钮）
- 恢复之前的备份（第一个按钮）

使用指南：

1. 当遇到 Cursor TooMany 问题时，点击最右侧按钮进行重置
2. 建议在重置前使用备份按钮备份数据
3. 如需恢复之前的数据，使用第一个按钮从最后一次备份中恢复
4. 定期使用刷新按钮更新数据，保持正常运行

该工具帮助管理 Cursor 应用程序的 Storage.json 数据。
